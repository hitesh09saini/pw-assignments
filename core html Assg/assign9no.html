Most common text formatting tags in HTML

  1. < strong> tag:
    • Use cases:
      • To emphasize text and give it strong importance.
      • Used for conveying importance to search engines and assistive technologies for accessibility.
    • Best practices:
      • Avoid using < strong> for purely stylistic purposes; use CSS for styling.
      • Use it only to indicate strong emphasis on the content.
  2. < em> tag:
    • Use cases:
      • To emphasize text and give it subtle importance.
      • Used to convey stress or voice inflection in a sentence.
    • Best practices:
      • Avoid using < em> for purely stylistic purposes; use CSS for styling.
      • Use it only to indicate emphasis or changes in the tone of the content
  3. < u> tag:
    • Use cases:
      • To underline text, traditionally used for hyperlinks.
      • Used to indicate that text is a non-hyperlink special reference.
    • Best practices:
      • Avoid using < u> for hyperlinks since it may cause confusion with standard underlined links.
      • Use it sparingly and only when necessary to maintain clarity.
  4. < del> tag:
    • Use cases:
      • To represent deleted or removed text, often used in revision history or strike-throughs.
    • Best practices:
      • Use it in combination with the < ins> tag to represent text changes accurately.
      • Avoid using < del> solely for stylistic strike-through effects; use CSS for that purpose.
  5. < ins> tag:
    • Use cases:
      • To represent inserted or added text, often used in revision history or showing new content.
    • Best practices:
      • Use it in combination with the < del> tag to represent text changes accurately.
      • Avoid using < ins> solely for stylistic underline effects; use CSS for that purpose.
  6. < sup> tag:
    • Use cases:
      • To display superscript text, commonly used for mathematical notations and footnotes.
    • Best practices:
      • Use it specifically for content that requires superscript, not for styling purposes.
      • Ensure the text size and position are adjusted appropriately for readability.
  7. < sub> tag:
    • Use cases:
      • To display subscript text, often used in chemical formulas and mathematical expressions.
    • Best practices:
      • Use it specifically for content that requires subscript, not for styling purposes.
      • Ensure the text size and position are adjusted appropriately for readability.
  8. < blockquote> tag:
    • Use cases:
      • To indicate a block of quoted content from an external source.
    • Best practices:
      • Use it when quoting content longer than a few lines to create clear demarcation.
      • Pair it with appropriate citation and use CSS for any additional styling.
  9. < code> tag:
    • Use cases:
      • To display inline code snippets or programming code.
    • Best practices:
      • Use it for short code snippets or single elements within a sentence.
      • For larger code blocks or multiline code, consider using < pre> or code blocks in < pre> tags.
  10. < kbd> tag:
    • Use cases:
      • To represent user keyboard input or key names.
    • Best practices:
      • Use it to indicate actual keyboard input, not for generic text formatting.
      • Combine it with other elements like < code> to represent keyboard shortcuts.

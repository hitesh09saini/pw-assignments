HTML TAGS

  1. < html > :

    • Description: The root element of an HTML document that wraps all other elements.
    • Usage: It defines the beginning and end of an HTML document.

  2. < head > :

    • Description: Contains meta-information about the HTML document, such as title, links to stylesheets, scripts, etc.
    • Usage: It is used to specify elements that are not displayed on the web page but provide important information to the browser and search engines.

  3. < title > :

    • Description: Sets the title of the HTML document that appears in the browser's title bar or tab.
    • Usage: It helps in identifying the page and improves SEO as search engines use the title to display search results.

  4. < body > :

    • Description: Contains the visible content of the HTML document.
    • Usage: All the content that users see on the web page, including text, images, links, and multimedia, is placed within the < body > tag.

  5. < h1 > - < h6>:

    • Description: Heading elements that define headings of different levels.
    • Usage: < h1> represents the highest level of the heading, and < h6> represents the lowest level. They are used for hierarchical structuring of the content.

  6. < p > :

    • Description: Represents a paragraph of text.
    • Usage: It is used to group text content into paragraphs, making the content more organized and readable.

  7. < a > :

    • Description: Creates a hyperlink to another web page or resource.
    • Usage: It is used to link to other pages, files, or locations within the same page.

  8. < img > :

    • Description: Embeds an image in the HTML document.
    • Usage: It is used to display images on the web page. Requires the src attribute to specify the image URL.

  9. < a > :

    • Description: Creates a hyperlink to another web page or resource.
    • Usage: It is used to link to other pages, files, or locations within the same page.

  10. < ul > and < li> :

    • Description: < ul> represents an unordered list, and < li> represents list items within the list.
    • Usage: < ul> is used to create bulleted lists, and each item is denoted by < li>.

  11. < ol > and < li> :

    • Description: < ol> represents an ordered list, and < li> represents list items within the list.
    • Usage: < ol> is used to create numbered lists, and each item is denoted by < li>.

  12. < div > :

    • Description: A generic container that does not convey any semantic meaning on its own.
    • Usage: It is used for grouping and styling purposes and helps structure the layout of the web page.

  13. < span > :

    • Description: Inline container that does not convey any semantic meaning on its own.
    • Usage: It is used for styling specific portions of text or inline elements within a block-level element.

  14. < table > :

    • Description: Creates a table for tabular data.
    • Usage: It is used to display data in rows and columns, and it contains < tr> (table row), < th> (table header), and < td> (table data) elements.

  15. < form > :

    • Description: Defines a form that contains input elements for user interaction.
    • Usage: It is used to collect user input, such as text fields, checkboxes, radio buttons, etc., and submit it to a server for processing.

  16. < input > :

    • Description: Represents an input field where users can enter data.
    • Usage: It is used with various type attributes (e.g., text, checkbox, radio) to create different types of input fields.

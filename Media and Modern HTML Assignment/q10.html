HTML vs HTML5

HTML HTML5
HTML is the older version of the HyperText Markup Language. HTML5 is the latest version of the HyperText Markup Language.
HTML is based on SGML (Standard Generalized Markup Language). HTML5 is not based on SGML; it is based on a new parsing algorithm called "HTML Living Standard."
HTML does not support audio and video embedding directly. HTML5 introduces native support for audio and video elements, allowing seamless multimedia embedding.
